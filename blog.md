## Svojstva
* registriranje i logiranje
* logirani korisnici mogu kreirati i uređivati članke
* naslovnica se sastoji od popisa članaka (ugrađeni Laravel pagination)
* svaki članak ima svoj link
* REST API dizajn
* responsive web dizajn
